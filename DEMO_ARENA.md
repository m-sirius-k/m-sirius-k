# MoCKA Demo Arena

This is a temporary demo venue for evaluators.

Goal
Show an interactive, browser-only demonstration of the Research Gate verification.

---

## Demo 1: Interactive Research Gate Verification (Browser)

What you do
Open the workflow page and click "Run workflow".

What happens
GitHub Actions runs the Research Gate verification in a clean environment.

What you can verify
The logs include the final verdict, including "RESEARCH_RUN: OK" when the ecosystem verification passes.

Demo link
https://github.com/nsjpkimura-del/MoCKA/actions/workflows/research-gate-demo.yml
